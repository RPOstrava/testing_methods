9 běžně používaných testů

  • Test funkčnosti tlačítka

    • Tento test ověřuje, zda tlačítko provádí požadovanou akci nebo reakci při stisknutí. Příklad: Pokud máte tlačítko "Odeslat formulář", testovali byste, zda se formulář skutečně odesílá po kliknutí na toto tlačítko.

  • Test validace formuláře

    • Tento test zjišťuje, zda formulář správně validuje vstupy uživatele podle předem stanovených pravidel. Příklad: Když vyplníte e-mailovou adresu ve formuláři, testovali byste, zda formulář odmítne neplatnou e-mailovou adresu, například bez @ znaku.

  • Test ověření textu

    • Tento test kontroluje správnost textu na stránce nebo v aplikaci. Příklad: Pokud máte na webové stránce titulek "Vítejte na našem webu", testovali byste, zda se tento text skutečně zobrazuje na požadovaném místě.

  • Test načasování

    • Tento test ověřuje, zda jsou časové aspekty aplikace nebo stránky v souladu s očekáváním. Příklad: Když máte načasovanou událost, například zobrazení reklamy po 30 sekundách, testovali byste, zda se reklama zobrazí po uplynutí tohoto času.

  • Test navigace

    • Tento test se zaměřuje na ověření správnosti navigace uživatelů po stránce nebo v aplikaci. Příklad: Pokud máte navigační menu s položkami "Domů", "O nás" a "Kontakt", testovali byste, zda se při kliknutí na tyto položky přesunete na příslušné stránky.

  • Test kompatibility prohlížeče

    • Tento test ověřuje, zda je webová stránka nebo aplikace správně zobrazována a funkční na různých prohlížečích. Příklad: Testovali byste zobrazení a funkčnost stránky ve webových prohlížečích jako Chrome, Firefox, Safari atd.

  • Test API

    • Tento test ověřuje, zda rozhraní API funguje správně a odpovídá specifikacím. Příklad: Pokud máte API pro získání seznamu uživatelů, testovali byste, zda volání tohoto API vrací očekávané údaje ve správném formátu.

  • Test webové služby

    • Tento test ověřuje funkčnost a integritu webové služby, která může poskytovat různé funkce nebo data. Příklad: Když máte webovou službu pro získání aktuálního počasí, testovali byste, zda tato služba vrací přesné informace o počasí na základě zadaných parametrů.

  • Test zabezpečení

    • Tento test ověřuje, zda je aplikace nebo stránka chráněna před různými bezpečnostními hrozbami a útoky. Příklad: Testovali byste například, zda je aplikace odolná proti SQL injection útokům nebo cross-site scripting (XSS) útokům.
